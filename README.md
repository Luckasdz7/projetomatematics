# ProjetoMatematics 🧮

![C#](https://img.shields.io/badge/c%23-%23239120.svg?style=for-the-badge&logo=c-sharp&logoColor=white)
![.NET](https://img.shields.io/badge/.NET-5C2D91?style=for-the-badge&logo=.net&logoColor=white)

Um aplicativo desktop (Windows Forms) focado em cálculos matemáticos, incluindo uma calculadora, resolutor de funções e um cronômetro.

Projeto feito para melhorar e desenvolver conhecimentos básicos de C# no Visual Studio.

---


## ✨ Funcionalidades Principais

O projeto é dividido em três módulos principais, acessíveis a partir de um menu inicial:

1.  **Calculadora**
    * Operações matemáticas básicas:
        * Adição
        * Subtração
        * Multiplicação
        * Divisão
    * Botão para limpar os campos de valor e resultado.

2.  **Funções Simples**
    * **Função de 1º Grau:** Resolve equações no formato `Ax + B = Y`, encontrando o valor de X.
    * **Função de 2º Grau:** Resolve equações no formato `Ax² + Bx + C = 0` utilizando a Fórmula de Bhaskara para encontrar os valores de X1 e X2.

3.  **Cronômetro**
    * Permite Iniciar, Continuar e Congelar (Pausar) a contagem de tempo.
    * Função para Restaurar (Zerar) o cronômetro.

## 🛠️ Tecnologias Utilizadas

* **C#**
* **.NET Framework 4.7.2**
* **Windows Forms (WinForms)** - para a interface gráfica.

## 🚀 Como Executar o Projeto

Para executar este projeto, você precisará do Visual Studio e do .NET Framework 4.7.2.

1.  Clone este repositório:
    ```sh
    git clone https://github.com/Luckasdz7/projetomatematics.git
    ```
2.  Abra o arquivo da solução `projetomatematics.sln` no Visual Studio.
3.  Pressione `F5` ou clique no botão "Iniciar" para compilar e executar o projeto.
